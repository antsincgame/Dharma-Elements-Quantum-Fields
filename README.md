## 🚀 **Quick Start**

### Prerequisites
- Modern web browser with **WebGL support**
- **Internet connection** for Three.js library
- **Open mind** for quantum-dharma understanding

### Installation
```bash
# Clone the repository
git clone https://github.com/antsincgame/Dharma-Elements-Quantum-Fields.git
cd Dharma-Elements-Quantum-Fields

# Start local server
python3 -m http.server 8000

# Open in browser
open http://localhost:8000/src/index.html
```

### Usage
1. **🧘‍♂️ Observe** the sacred mandala visualization
2. **🖱️ Hover** over elements for deep wisdom
3. **🎛️ Adjust** dharma parameters with sliders
4. **🎵 Click** meditation buttons for mantras/sutras
5. **🌊 Flow** with the interdependent dance of reality
